PRE-INSTALLATION
----------------
- create DOTCLOUD account
- install DOTCLOUD CLI

INSTALLATION
------------
- unzip JAP_WS_REMOTE_NODE_DOTCLOUD-X.X.X.zip
- open JAP_WS_REMOTE_NODE_DOTCLOUD-X.X.X
- configure JAP_WS_REMOTE.json
- open CONSOLE
	- cd JAP_WS_REMOTE_NODE_DOTCLOUD-X.X.X
	- dotcloud create application-id
	- dotcloud push application-id

---------
